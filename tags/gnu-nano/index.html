更改 git 默认编辑器

 2023-06-04  345 字  git vi/vim GNU nano 

在 git commit 的时候，默认的编辑器是 GNU nano，但对于重度 vim 用户来说，还是 vim 用的顺手(x)

这时候可以通过 git config --global core.editor vi 来修改全局默认的 commit 编辑器
